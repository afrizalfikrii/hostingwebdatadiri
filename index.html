About Me

Saya adalah mahasiswa Teknik Informatika di Universitas Muhammadiyah Surakarta. Saya memiliki ketertarikan besar dalam dunia teknologi, khususnya dalam pengembangan web dan pemrograman. Saya senang mempelajari hal-hal baru di bidang IT dan selalu berusaha untuk mengembangkan keterampilan saya dalam coding dan problem-solving.

Walaupun saya belum memiliki pengalaman kerja, saya aktif dalam berbagai proyek pribadi dan pembelajaran mandiri. Saya percaya bahwa teknologi memiliki potensi besar untuk mengubah dunia, dan saya ingin menjadi bagian dari perubahan tersebut. Tujuan karier saya adalah menjadi seorang pengembang perangkat lunak yang dapat menciptakan solusi inovatif untuk berbagai permasalahan.

Saya juga memegang prinsip bahwa belajar adalah proses seumur hidup, dan kegigihan serta semangat untuk terus berkembang adalah kunci kesuksesan. Saya selalu berusaha untuk tetap produktif, disiplin, dan terus mengasah keterampilan saya di dunia teknologi.
